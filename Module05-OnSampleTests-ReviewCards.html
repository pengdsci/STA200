One‑Sample Tests for Location

📚 Reference Cards: t‑test • Sign Test • Regression Approach
🧪 One‑Sample t‑Test parametric
\[ H_0: \mu = \mu_0 \quad \text{vs} \quad H_a: \mu \neq \mu_0 \] \[ TS = \frac{\bar{y} - \mu_0}{s / \sqrt{n}} \;\sim\; t_{n-1} \] \[ p\text{-value} = 2 \cdot P(t_{n-1} \ge |TS|) \]
# manual calculation (R as calculator) BMI = PimaIndiaDiabetes$mass n = length(BMI) xbar = mean(BMI) s = sd(BMI) mu0 = 30 TS = (xbar - mu0) / (s / sqrt(n)) pvalue = 2 * pt(abs(TS), df = n-1, lower.tail = FALSE) cbind(xbar, TS, df = n-1, pvalue) # built-in t.test() t.test(x = BMI, alternative = "two.sided", mu = 30, conf.level = 0.95)
🔄 Regression View GLM unification
\[ y = \beta_0 + \epsilon, \quad H_0: \beta_0 = \mu_0 \] \[ \text{Transform: } y - \mu_0 = \alpha_0 + \epsilon,\quad H_0:\alpha_0 = 0 \] \[ TS = \frac{\hat{\alpha}_0}{se(\hat{\alpha}_0)} \sim t_{n-1} \]
# Equivalent t-test via linear model (intercept only) # using I() to shift response one.sample.t.reg = lm(I(BMI - 30) ~ 1, data = PimaIndiaDiabetes) summary(one.sample.t.reg) # identical t-value and p-value as t.test() # estimate = mean(BMI) - 30 = 3.086, t = 8.695
✍️ Sign Test (Median) distribution‑free
\[ H_0: M = M_0 \quad\text{vs}\quad H_a: M \neq M_0 \] \[ D_i = Y_i - M_0,\quad S = \#\{D_i > 0\} \] \[ S \sim \text{Binomial}(n, p=0.5),\quad n = \text{non-zero diffs} \] \[ \text{If } n\ge 25:\quad Z = \frac{S - 0.5n}{0.5\sqrt{n}} \approx N(0,1) \]
# manual sign test example (reaction times) reaction.time = c(230,245,252,248,255,260,235,240,250,242,238,258,245,248,253) pos.sign = sum(reaction.time - 250 > 0) neg.sign = sum(reaction.time - 250 < 0) n = pos.sign + neg.sign pval = 2 * pbinom(min(pos.sign, neg.sign), size = n, prob = 0.5) pval # exact binomial test binom.test(x = pos.sign, n = n, p = 0.5, alternative = "two.sided") # using BSDA::SIGN.test (if installed) # install.packages("BSDA"); library(BSDA) # SIGN.test(reaction.time, md = 250)
📊 Binomial Tools p‑values & quantiles
\[ P(X = k) = \binom{n}{k} p^k (1-p)^{n-k},\quad p=0.5 \] \[ \text{Sign test p-value (two-tailed)} = 2 \times P(S \le \min(s, n-s)) \]
# binomial probabilities & quantiles # P(X > 3) with n=10, p=0.5 pbinom(3, size = 10, prob = 0.5, lower.tail = FALSE) # 90th percentile for Binom(20,0.5) qbinom(0.9, size = 20, prob = 0.5) # two-tailed sign test p-value manual: s_obs = 5; n_trials = 14 p_binomial = 2 * pbinom(5, size = 14, prob = 0.5) p_binomial
🧬 Real‑Data Workflow BMI obesity test
\[ H_0: \mu = 30 \;(\text{not obese})\quad vs \quad H_a: \mu \neq 30 \] \[ \text{BMI data: } \bar{x}=33.09,\; s=7.09,\; n=392,\; t=8.695,\; p \approx 0 \]
# load data and one-sample t-test Pima = read.csv("https://pengdsci.github.io/STA200/dataset/PimaIndiaDiabetes.csv") BMI = Pima$mass t.test(BMI, mu = 30, alternative = "two.sided") # Conclusion: Reject H0 → evidence that true mean BMI ≠ 30 # 95% CI: (32.39, 33.78)
📌 Sign Test in Action median reaction time
\[ H_0: M = 250 \quad vs \quad H_a: M \neq 250 \] \[ \text{positive signs} = 5,\; \text{negative}=9,\; n=14 \] \[ p\text{-value} = 2\times P(S\le 5) = 0.424 \]
# reaction time example (from slides) rt = c(230, 245, 252, 248, 255, 260, 235, 240, 250, 242, 238, 258, 245, 248, 253) SIGN.test = if(require(BSDA, quietly = TRUE)) { BSDA::SIGN.test(rt, md = 250) } else { cat("Install BSDA for full median CI\n") } # Using binom.test directly pos = sum(rt > 250); n_nonzero = sum(rt != 250) binom.test(pos, n_nonzero, p = 0.5, conf.level = 0.95)
⚖️ t‑test vs Sign Test choice guide
\[ \textbf{t-test (mean)}: \text{Normal dist or large n} \quad \] \[ \textbf{Sign test (median)}: \text{Any distribution, robust} \]
# t-test more powerful when normality holds # sign test more robust to outliers and heavy tails # Regression view: t-test = intercept model # Sign test = binomial proportion test on signs
