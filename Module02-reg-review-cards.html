📌 Key inferential procedures, formulas & R code
🔬 t-test for slope \( \beta_1 \) H₀: β₁ = 0
\[ H_0: \beta_1 = 0 \quad \text{vs} \quad H_a: \beta_1 \neq 0 \] \[ TS = \frac{b_1 - 0}{SE(b_1)} \;\sim\; t_{n-2} \]
# Fit linear model (using "=" assignment) model = lm(grade ~ study_hours, data = df) # Extract t-statistic from summary summary(model) # Manual computation b1 = coef(model)[2] se_b1 = sqrt(diag(vcov(model)))[2] t_stat = b1 / se_b1 # Two-tailed p-value (n = sample size) n = nrow(df) p_value = 2 * pt(abs(t_stat), df = n-2, lower.tail = FALSE)
✓ Reviewed t‑test procedure & formula
📐 Confidence Interval for \( \beta_1 \) 95% CI
\[ b_1 \;\pm\; t_{\alpha/2,\, df=n-2} \times SE(b_1) \] \[ \text{Critical value: } t_{0.975,\, n-2} \]
# Built-in CI confint(model, parm = "study_hours", level = 0.95) # Manual calculation b1 = coef(model)[2] se = sqrt(diag(vcov(model)))[2] crit = qt(0.975, df = df.residual(model)) ci_lower = b1 - crit * se ci_upper = b1 + crit * se
📏 CI construction & critical value (t-dist)
📈 Estimated model & residuals \(\hat{Y} = b_0 + b_1 X\)
\[ \hat{Y}_i = b_0 + b_1 X_i, \quad e_i = Y_i - \hat{Y}_i \] \[ RSS = \sum_{i=1}^{n} e_i^2 \]
# Extract fitted values and residuals fitted_vals = fitted(model) residuals_vals = resid(model) # Diagnostic plots plot(model, which = 1) # Residuals vs Fitted qqnorm(resid(model)) qqline(resid(model), col = "#6E3061")
🔍 Residual extraction & diagnostic plots
⚙️ Standard Error \(SE(b_1)\) inference core
\[ SE(b_1) = \frac{\hat{\sigma}}{\sqrt{\sum (X_i - \bar{X})^2}}, \quad \hat{\sigma} = \sqrt{\frac{RSS}{n-2}} \]
# Extract SE(b1) directly se_b1 = summary(model)$coefficients[2, "Std. Error"] # Residual standard error (sigma hat) sigma_hat = summary(model)$sigma
📌 SE formula & residual std error σ̂
📊 R² & F-test model fit & slope test
\[ R^2 = \frac{SS_{reg}}{SS_{tot}} = 1 - \frac{RSS}{SS_{tot}} \] \[ F = \frac{SS_{reg}/1}{RSS/(n-2)} \sim F_{1,\, n-2} \]
# R-squared and F-statistic summary(model)$r.squared summary(model)$fstatistic # For SLR: F = t_stat^2 t_stat = coef(summary(model))[2, "t value"] f_stat = t_stat^2
📈 R² interpretation & F-test relation to t-test
🎯 Influence diagnostics Cook's Distance
\[ D_i = \frac{e_i^2}{p \cdot MSE} \cdot \frac{h_{ii}}{(1-h_{ii})^2} \] \(h_{ii}\) = leverage, \(p\) = #parameters (2 in SLR)
# Cook's distance cooksD = cooks.distance(model) plot(cooksD, type = "h", main = "Cook's Distance") # Leverage values leverage = hatvalues(model) # Rule of thumb: D_i > 4/n is influential
⚠️ Detect outliers & high-leverage points
🧪 Normality of residuals QQ-plot & test
\[ \varepsilon_i \sim N(0,\sigma^2) \quad \text{(required for inference)} \]
# Q-Q plot visual check qqnorm(resid(model), main = "Normal Q-Q Plot") qqline(resid(model), col = "#6E3061", lwd = 2) # Shapiro-Wilk test for normality shapiro.test(resid(model))
✅ Check normality assumption before inference
🌊 Homoscedasticity constant error variance
\[ \text{Var}(\epsilon_i \mid X) = \sigma^2 \quad \forall i \]
# Residuals vs Fitted plot plot(model, which = 1, main = "Residuals vs Fitted") # Breusch-Pagan test (requires lmtest package) # install.packages("lmtest") library(lmtest) bptest(model)
📉 Detect heteroscedasticity (fan-shaped pattern)
🔮 Prediction & Confidence intervals new X
\[ \hat{Y}_h \pm t_{\alpha/2,n-2} \times SE(\hat{Y}_h) \quad\text{(mean response)} \] \[ \hat{Y}_h \pm t_{\alpha/2,n-2} \times \sqrt{MSE + [SE(\hat{Y}_h)]^2} \quad\text{(prediction)} \]
# Create new data new_data = data.frame(study_hours = c(8, 10)) # Confidence interval for mean response predict(model, newdata = new_data, interval = "confidence") # Prediction interval for individual observation predict(model, newdata = new_data, interval = "prediction")
📦 Prediction for new observations
⚡ Leverage & Studentized residuals outlier detection
\[ h_{ii} = \frac{1}{n} + \frac{(X_i - \bar{X})^2}{\sum (X_j - \bar{X})^2} \] \[ \text{Studentized residual} = \frac{e_i}{\hat{\sigma}\sqrt{1-h_{ii}}} \]
# Leverage values leverage = hatvalues(model) # Rule of thumb: high leverage > 2p/n = 4/n # Externally studentized residuals rstudent(model) # Identify outliers: |rstudent| > 2 or 3
🎯 Identify high leverage & influential outliers
📉 p-value calculation two-tailed test
\[ p\text{-value} = 2 \times P(T_{n-2} > |t_{obs}|) \]
# Given t-statistic and degrees of freedom t_obs = 24.02 # example from course grade data df = 8 # n - 2 = 10 - 2 # Compute two-tailed p-value p_value = 2 * pt(abs(t_obs), df = df, lower.tail = FALSE) # Output: ~9.62e-09 (highly significant)
🧮 Exact p-value from t-distribution
📉 Linearity check scatter plot + residual pattern
\[ E[Y \mid X] = \beta_0 + \beta_1 X \]
# Scatter plot with regression line plot(x = df$study_hours, y = df$grade, main = "Scatter Plot with Regression Line", xlab = "Study Hours", ylab = "Course Grade") abline(model, col = "#6E3061", lwd = 2) # Check residuals vs fitted for non-linearity plot(model, which = 1)
📌 Visual inspection for linear relationship
