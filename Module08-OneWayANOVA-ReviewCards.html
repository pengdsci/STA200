📋 ANOVA & CRD Analysis

🔬ANOVA & CRD Analysis: Completely Randomized Design · F‑test · Tukey HSD · Bonferroni
📐 F‑Test for CRD H₀: μ₁=...=μₖ
\[ SS_T = SS_B + SS_W,\quad MS_B = \frac{SS_B}{k-1},\quad MS_W = \frac{SS_W}{n_T - k} \] \[ F = \frac{MS_B}{MS_W} \;\sim\; F_{(k-1,\; n_T - k)} \]
# manual ANOVA from summary statistics t = 5 # number of groups n_T = 50 # total observations dfN = t - 1 dfD = n_T - t SS_B = 1077.944 SS_W = 236.300 MS_B = SS_B / dfN MS_W = SS_W / dfD F_value = MS_B / MS_W p_value = pf(F_value, dfN, dfD, lower.tail = FALSE) # using aov() long format aov.model = aov(Length ~ Treatment, data = pea.long) summary(aov.model)
📦 One‑Way ANOVA in R aov() + assumptions
Assumptions: independence, normality (Shapiro–Wilk), homogeneity of variances (Levene/Bartlett).
\[ \text{Residuals: } e_{ij} = y_{ij} - \bar{y}_{i\cdot} \]
# long format example (pea sections) lengths = c(75,67,70,75,65,71,67,67,76,68, # Control 57,58,59,59,62,60,60,57,59,61, # Glucose 58,61,56,58,57,56,58,60,57,58, # Fructose 58,59,58,61,57,56,58,57,57,59, # GlucFruc 62,66,65,63,64,62,65,65,62,67) # Sucrose treatment = rep(c("Control","Glucose","Fructose","GlucFruc","Sucrose"), each=10) pea.long = data.frame(Length = lengths, Treatment = factor(treatment)) # run ANOVA aov.model = aov(Length ~ Treatment, data = pea.long) anova(aov.model) # or summary(aov.model)
🔍 Tukey HSD family‑wise error rate
\[ \text{HSD} = q_{\alpha, k, df_W} \times \sqrt{\frac{MS_W}{n}} \] Pairwise confidence intervals: \( \bar{y}_i - \bar{y}_j \pm \text{HSD} \)
# after significant ANOVA aov.model = aov(Length ~ Treatment, data = pea.long) tukey.comp = TukeyHSD(aov.model) print(tukey.comp) # plot the comparisons par(mai = c(1.5,2,1,1)) plot(tukey.comp, las = 1, cex.lab = 0.6)
🛡️ Bonferroni Adjustment conservative p‑value correction
\[ p_{\text{adj}} = \min(m \times p_{\text{raw}},\; 1) \] where \( m \) = number of comparisons. Controls family‑wise error.
# pairwise t-tests with Bonferroni Bonferroni.result = pairwise.t.test( x = pea.long$Length, g = pea.long$Treatment, p.adjust.method = "bonferroni" ) print(Bonferroni.result) # alternative: adjust p-values manually pairwise.t.test(pea.long$Length, pea.long$Treatment, p.adjust = "bonferroni")

🧮 SS Decomposition & ANOVA Table SSB, SSW, SST
\[ \underbrace{\sum\sum (y_{ij}-\bar{y}_{..})^2}_{SS_T}= \underbrace{\sum n_i(\bar{y}_{i.}-\bar{y}_{..})^2}_{SS_B}+ \underbrace{\sum\sum (y_{ij}-\bar{y}_{i.})^2}_{SS_W} \]
ANOVA table layout: \[ \begin{array}{l|ccc} \text{Source} & df & SS & MS & F \\ \hline \text{Between} & k-1 & SS_B & SS_B/(k-1) & MS_B/MS_W \\ \text{Within} & N-k & SS_W & SS_W/(N-k) & \\ \text{Total} & N-1 & SS_T & & \end{array} \]
# compute manually in R Control = c(75,67,70,75,65,71,67,67,76,68) Glucose = c(57,58,59,59,62,60,60,57,59,61) Fructose = c(58,61,56,58,57,56,58,60,57,58) GlucFruc = c(58,59,58,61,57,56,58,57,57,59) Sucrose = c(62,66,65,63,64,62,65,65,62,67) ybar_grand = mean(c(Control,Glucose,Fructose,GlucFruc,Sucrose)) SS_B = 10 * sum( (c(mean(Control),mean(Glucose),mean(Fructose),mean(GlucFruc),mean(Sucrose)) - ybar_grand)^2 ) SS_W = sum( (Control-mean(Control))^2 ) + sum( (Glucose-mean(Glucose))^2 ) + sum( (Fructose-mean(Fructose))^2 ) + sum( (GlucFruc-mean(GlucFruc))^2 ) + sum( (Sucrose-mean(Sucrose))^2 )
🎲 F‑Distribution Basics right‑skewed
\[ F = \frac{MS_B}{MS_W} \sim F_{df_1, df_2} \] Critical value: \(F_{\alpha, df_1, df_2}\).
If \(F > F_{\text{crit}}\) → reject \(H_0\). p‑value = \(P(F_{df1,df2} \geq F_{\text{obs}})\).
# compute p-value and critical value in R df1 = 4 # k-1 df2 = 45 # N-k F_obs = 51.32 p_val = pf(F_obs, df1, df2, lower.tail = FALSE) # ≈ 0 # critical value at alpha=0.05 crit_val = qf(0.95, df1, df2) # pf(crit_val, df1, df2, lower.tail=FALSE) = 0.05
🧪 Post‑hoc Summary Tukey HSD vs Bonferroni
After rejecting \(H_0\), use post‑hoc tests:
Tukey HSD : all pairwise comparisons, controls FWER.
Bonferroni : adjust \(\alpha^* = \alpha/m\), very conservative.
# from the pea example: conclusions # Tukey shows Sucrose differs from all except? Actually from output: # Glucose-Control, Fructose-Control, GlucFruc-Control, Sucrose-Control all significant. # Fructose-Glucose not significant, GlucFruc-Fructose not significant. # Bonferroni yields similar: Glucose vs Fructose p=1.0 (non-sig) # implementation: tukey_res = TukeyHSD(aov(Length ~ Treatment, data=pea.long)) bonf_res = pairwise.t.test(pea.long$Length, pea.long$Treatment, p.adjust="bonferroni") print("Tukey adjusted p-values:"); print(tukey_res$Treatment[,4]) print("Bonferroni p-values:"); print(bonf_res$p.value)

💡 MSB vs MSW intuition variance ratio
\[ MS_B = \frac{\sum n_i(\bar{y}_{i\cdot}-\bar{y}_{..})^2}{k-1}, \quad MS_W = \frac{\sum_i (n_i-1)s_i^2}{N-k} \] If \(H_0\) true: \(MS_B\) and \(MS_W\) estimate same \(\sigma^2\) → \(F \approx 1\).
# extracting MSB and MSW from aov object aov_out = aov(Length ~ Treatment, data = pea.long) aov_sum = summary(aov_out) MS_B = aov_sum[[1]]$"Mean Sq"[1] MS_W = aov_sum[[1]]$"Mean Sq"[2] F_ratio = MS_B/MS_W cat("F-ratio from model: ", F_ratio)
