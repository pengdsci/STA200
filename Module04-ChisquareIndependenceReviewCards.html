📊 Chi‑Square Independence & Risk Measures

Reference cards: contingency tables, expected frequencies, Risk Ratios, R Code
📐 χ² Test of Independence formula + R
\[ H_0: \text{Row variable independent of Column variable} \] \[ E_{ij} = \frac{(\text{row total}_i)(\text{col total}_j)}{\text{grand total}} \] \[ \chi^2 = \sum_{i=1}^{r}\sum_{j=1}^{c} \frac{(O_{ij} - E_{ij})^2}{E_{ij}} \sim \chi^2_{(r-1)(c-1)} \]
# Build contingency table (values by row) cell.vec = c(68, 106, 131, 40, 119, 103, 127, 81, 70, 52, 51, 52) cont.table = matrix(cell.vec, nrow = 3, byrow = TRUE) rownames(cont.table) = c("0-10", "11-20", "More Than 20") colnames(cont.table) = c("Humanities", "SocSci", "Business", "Engineering") # Perform chi-square test chisq.test(cont.table)
📌 Reject H0 if p-value ≤ α
⚠️ Risk Measures: RR & OR (2×2) cohort / case-control
\[ AR_{\text{exp}} = \frac{a}{a+b},\quad AR_{\text{unexp}} = \frac{c}{c+d} \] \[ RR = \frac{AR_{\text{exp}}}{AR_{\text{unexp}}} = \frac{a/(a+b)}{c/(c+d)},\qquad OR = \frac{a/b}{c/d} = \frac{ad}{bc} \] Interpretation: \(=1\) no association; \(>1\) risk factor; \(<1\) protective.
# Calculate RR and OR using "=" assignment a = 30; b = 10; c = 15; d = 25 risk_exp = a/(a+b) risk_unexp = c/(c+d) RR = risk_exp / risk_unexp OR = (a*d)/(b*c) print(paste("RR =", RR, "| OR =", OR))
📋 Expected Table Under H₀ step-by-step
Step 1: Marginal probabilities \(p_{i.}= \frac{\text{row total}_i}{n}\), \(p_{.j}= \frac{\text{col total}_j}{n}\)
Step 2: Under independence: \(\hat{p}_{ij} = p_{i.} \times p_{.j}\)
Step 3: Expected frequency \(E_{ij} = \hat{p}_{ij} \times n\)
# Attendance vs performance (2x2) observed = matrix(c(25, 2, 8, 15), nrow = 2, byrow = TRUE) rownames(observed) = c("Good", "Poor") colnames(observed) = c("Pass", "Fail") row_tot = rowSums(observed) col_tot = colSums(observed) n_total = sum(observed) expected = (row_tot %*% t(col_tot)) / n_total # quick check with chisq.test chisq.test(observed)$expected

📁 Study Design → Measure epidemiology
  • Cohort → Relative Risk (RR)
  • Case-Control → Odds Ratio (OR)
  • Cross-Sectional → Prevalence Ratio
  • RCT → Risk Ratio / OR
\[ \begin{array}{c|cc} & \text{Disease+} & \text{Disease-} \\ \hline \text{Exposed+} & a & b \\ \text{Exposed-} & c & d \end{array} \]
# Compute both measures from 2x2 matrix tab = matrix(c(45, 15, 30, 40), nrow = 2, byrow = TRUE) OR_val = (tab[1,1]*tab[2,2])/(tab[1,2]*tab[2,1]) RR_val = (tab[1,1]/(tab[1,1]+tab[1,2])) / (tab[2,1]/(tab[2,1]+tab[2,2])) cat("Odds Ratio =", OR_val, "\nRelative Risk =", RR_val)
🔍 Critical Value & p‑value decision rule
\[ df = (r-1)(c-1), \quad \text{Reject if } \chi^2 > \chi^2_{\alpha, df} \] \[ p\text{-value} = P(\chi^2_{df} \geq TS) \] Example (2×2): \(\chi^2_{0.05,1} \approx 3.841\)
# Critical value qchisq(0.95, df = 1) # 3.841459 # p-value from test statistic ts = 17.75 p_value = 1 - pchisq(ts, df = 1) # Tire quality example tire_data = matrix(c(106,124,123,67,85,115,37,72,112), nrow = 3, byrow = TRUE) chisq.test(tire_data)
