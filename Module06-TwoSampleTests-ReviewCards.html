🎴 Reference Cards: Two‑Sample Parametric & Nonparametric Tests

Pooled t‑test · Wilcoxon Rank Sum · Paired t‑test · Signed‑Rank Test
🧪 Pooled two‑sample t‑test parametric
\[ s_{\text{pool}} = \sqrt{\frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}} \quad,\quad T = \frac{\bar{x}_1 - \bar{x}_2}{s_{\text{pool}}\sqrt{\frac{1}{n_1}+\frac{1}{n_2}}} \sim t_{n_1+n_2-2} \]
# equal variances assumed, two independent samples n1 = length(group1); n2 = length(group2) xbar1 = mean(group1); xbar2 = mean(group2) s1 = sd(group1); s2 = sd(group2) s_pool = sqrt(((n1-1)*s1^2 + (n2-1)*s2^2) / (n1+n2-2)) TS = (xbar1 - xbar2) / (s_pool * sqrt(1/n1 + 1/n2)) p_value = 2 * pt(abs(TS), df = n1+n2-2, lower.tail = FALSE) # built-in t.test() t.test(group1, group2, alternative = "two.sided", var.equal = TRUE)
✔️ mastered: assumptions & pooling
📈 Regression = t‑test lm() equivalence
\[ Y_i = \beta_0 + \beta_1 \cdot \mathbb{I}_{\text{group2}} + \varepsilon_i \] t-test for \(\beta_1\) equals two‑sample pooled t‑statistic
# group variable must be a factor PimaIndiaDiabetes = read.csv("url_to_data") reg_model = lm(mass ~ factor(diabetes), data = PimaIndiaDiabetes) summary(reg_model) # coefficient for factor(diabetes)pos gives # mean difference & t-value identical to 2-sample t-test
✅ regression & t‑test linkage understood
🥇 Wilcoxon Rank Sum test non‑parametric
\[ U_1 = n_1 n_2 + \frac{n_1(n_1+1)}{2} - R_1,\quad U_2 = n_1 n_2 + \frac{n_2(n_2+1)}{2} - R_2 \] \[ U = \min(U_1, U_2),\quad Z = \frac{U - \frac{n_1 n_2}{2}}{\sqrt{\frac{n_1 n_2 (n_1+n_2+1)}{12}}} \approx N(0,1) \]
# independent samples, ordinal/not normal groupA = c(10,12,14,16); groupB = c(15,18,20,22) wilcox.test(groupA, groupB, exact = TRUE, alternative = "two.sided") # large sample approximation (n>20) wilcox.test(diabetes_BMI, healthy_BMI, exact = FALSE)
📌 H0: identical distributions
🔗 Paired sample t‑test within‑subjects
\[ t = \frac{\bar{d} - 0}{s_d / \sqrt{n}} \sim t_{n-1}, \quad d_i = \text{after}_i - \text{before}_i \]
# paired design: same subject measured twice before = c(150,160,145,155,140,165,152,158,148,162) after = c(140,155,142,150,138,160,148,152,145,156) t.test(after, before, paired = TRUE) # or t.test(differences) # equivalent: one-sample t-test on differences differences = after - before t.test(differences)
📋 normality of differences required
✍️ Wilcoxon Signed‑Rank paired / repeated
\[ \text{Rank } |d_i|,\ \text{sum ranks of positive/negative signs},\ W = \min(W^+, W^-) \] \[ \text{For } n > 20:\quad Z \approx \frac{W - n(n+1)/4}{\sqrt{n(n+1)(2n+1)/24}} \]
# pain relief example (paired, non-normal) before_pain = c(7,6,8,9,5,7,6,8,4,7) after_pain = c(5,6,7,5,4,8,3,9,2,6) wilcox.test(before_pain, after_pain, paired = TRUE, exact = FALSE) # exact p-value for small n: exact = TRUE wilcox.test(before, after, paired = TRUE, exact = TRUE)
🧠 H0: median difference = 0
📊 Applied case study BMI ~ diabetes status
\[ \text{Wilcoxon } W = 22608,\ p \approx 1.3\times 10^{-7} \] \[ t_{\text{pooled}} = 5.54,\ df=390,\ p = 5.56e-08 \]
Pima = read.csv("https://pengdsci.github.io/STA200/dataset/PimaIndiaDiabetes.csv") diab_BMI = Pima$mass[Pima$diabetes == "pos"] healthy_BMI = Pima$mass[Pima$diabetes == "neg"] # t-test (equal var) t.test(diab_BMI, healthy_BMI, var.equal = TRUE) # Wilcoxon rank sum (normal approx) wilcox.test(diab_BMI, healthy_BMI, exact = FALSE) # Both reject H0: BMI distributions differ significantly
⭐ both tests converge: strong evidence
