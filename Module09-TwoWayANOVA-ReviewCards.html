📊 Two-Way ANOVA & Post-Hoc Analysis

Reference Cards | R Code & Key Formulas: Replicated RBD • Main Effects • Interaction • Tukey HSD
📐 Hypotheses & F-Statistics core inferential
\[ H_0^{\text{A}}: \text{All levels of Factor A have same mean} \] \[ \quad\text{vs}\quad \] \[ H_a: \text{At least one differs} \] \[ F_A = \frac{MS_A}{MSE} \;\sim\; F_{(a-1),\, (N-ab)} \] \[ F_B = \frac{MS_B}{MSE} \;\sim\; F_{(b-1),\, (N-ab)} \] \[ F_{AB} = \frac{MS_{AB}}{MSE} \;\sim\; F_{(a-1)(b-1),\, (N-ab)} \] \(a\) = levels of factor A, \(b\) = levels factor B, \(N\) total sample size
# One-way ANOVA & Two-way ANOVA structure # For two-way with interaction: model = aov(response ~ factorA * factorB, data = mydata) summary(model) # Compute p-value from F (manual): p_value = pf(F_value, df1, df2, lower.tail = FALSE)
📋 ANOVA Table Breakdown SS decomposition
\[ SS_T = SS_A + SS_B + SS_{AB} + SS_E \] \[ df_T = N-1,\; df_A = a-1, \] \[ df_B = b-1, \] \[ df_{AB} = (a-1)(b-1), \] \[ df_E = N - ab \] \[ MS = \frac{SS}{df},\quad F = \frac{MS_{\text{effect}}}{MS_E} \]
# Construct ANOVA table in R growth.aov = aov(growth ~ sunlight * watering, data = growthData) summary(growth.aov) # Output interpretation: # Df, Sum Sq, Mean Sq, F value, Pr(>F)
⚙️ ANOVA Assumptions validation
  • Normality within each group combination
  • Homogeneity of variances (Homoscedasticity)
  • Independence of observations
  • Random sampling
# Check homogeneity (Levene's test) library(car) leveneTest(response ~ interaction(factorA, factorB), data = mydata) # Q-Q plot for normality check residuals = residuals(model) qqnorm(residuals); qqline(residuals, col = "red")
🔎 Post-hoc Tests & Tukey HSD multiple comparisons
\[ \text{Tukey's HSD: } \; \text{HSD} = q_{\alpha,k,df_E}\sqrt{\frac{MSE}{n}} \] Family-wise error rate controlled at \(\alpha=0.05\).
# Tukey HSD for main effects & interactions model.int = aov(response ~ factorA * factorB, data = df) TukeyHSD(model.int) # For specific factor (e.g., sunlight) : TukeyHSD(model.int, which = "factorA") # Visualize confidence intervals plot(TukeyHSD(model.int))

\[ \text{CRD vs RBD: } SS_T = SS_{Treatments}+SS_{Blocks}+SS_E \]
📈 Interaction Visualization graphical diagnostics
# Aggregate means for each combination grp.means = aggregate(growth ~ sunlight + watering, data = growthData, FUN = mean) # Extract Daily and Weekly means daily.avg = grp.means[grp.means$watering == "Daily", "growth"] weekly.avg = grp.means[grp.means$watering == "Weekly", "growth"] # Base line plot plot(1:4, weekly.avg, type = "b", col = "red", xaxt = "n", xlab = "Sunlight", ylab = "Mean Growth") lines(1:4, daily.avg, type = "b", col = "blue") axis(1, at = 1:4, labels = c("None","Low","Medium","High")) legend("topright", legend = c("Weekly","Daily"), col = c("red","blue"), lty = 1)
\[ \text{Parallel lines} \Rightarrow \text{No interaction}; \quad \text{Crossing} \Rightarrow \text{Interaction} \]
🧬 Interaction Effect Analysis simple comparisons
When interaction is significant: effect of Factor A depends on level of Factor B. \[ H_0: \text{No interaction} \quad \text{vs} \quad H_a: \text{Interaction present} \]
# If interaction significant: pairwise comparisons within each level # Example: compare sunlight effect for Daily and Weekly separately library(emmeans) emm = emmeans(model.int, ~ sunlight | watering) pairs(emm) # Or Tukey across all combinations TukeyHSD(model.int, which = "sunlight:watering")
📝 Main Effects & Reporting write-up style
Format 1: Two-way ANOVA revealed a significant main effect of A (F(df1, df2) = F-value, p = ...).
Format 2: No significant interaction (F(df1, df2) = , p > 0.05) → interpret main effects independently.
# Extract exact p-values from aov summary summary_aov = summary(model) pval_A = summary_aov[[1]]$`Pr(>F)`[1] # Reporting example: # "Sunlight had significant effect on growth (F(3,32)=23.05, p<.001)"
