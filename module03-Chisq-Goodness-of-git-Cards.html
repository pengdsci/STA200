STA 200 STATISTICS II

📊 Module 03: Chi-square Goodness-of-Fit

Reference cards: hypothesis formulation, test statistic, degrees of freedom, R implementation & p-value interpretation

📐 1. Hypotheses & Expected Counts

\[ H_0: P_1 = p_1, P_2 = p_2, \dots, P_k = p_k \] \[ H_a: \text{At least one } P_i \neq p_i \]

Expected frequencies: \(E_i = N \times p_i\)
where \(N\) = total sample size, \(p_i\) = hypothesized proportion.

# Expected counts from hypothesized proportions observed = c(O1, O2, ..., Ok) probs = c(p1, p2, ..., pk) N = sum(observed) expected = N * probs
✅ \(k\) = number of categories. df = \(k-1\). All \(\chi^2\) tests are right-tailed.

📈 2. Test Statistic (\(G^2\))

\[ G^{2} = \sum_{i=1}^{k} \frac{(O_i - E_i)^2}{E_i} \;\sim\; \chi^2_{(k-1)} \]

Interpretation: Larger \(G^2\) → stronger evidence against \(H_0\). p-value = \(P(\chi^2_{df} \ge G^2)\).

# Manual calculation in R (using = assignment) G.sq = sum((observed - expected)^2 / expected) df = length(observed) - 1 p_value = pchisq(G.sq, df = df, lower.tail = FALSE)
💡 Always right-tailed: use lower.tail = FALSE for p-value.

⚙️ 3. Built-in Test: chisq.test()

\[ \text{chisq.test(x, p = expected\_props)} \]

Direct goodness-of-fit test from base R {stats}. Returns \(G^2\), df, p-value.

# M&M example: observed counts and hypothesized probabilities observed = c(52, 48, 38, 22, 20, 20) prob_model = c(0.30, 0.20, 0.20, 0.10, 0.10, 0.10) test_result = chisq.test(observed, p = prob_model) print(test_result) # output: X-squared = 2.9667, df = 5, p-value = 0.7051
🔍 Ensure p = argument sums to 1. Otherwise R normalizes automatically.

🎯 4. Critical Value & Rejection Rule

\[ \text{Reject } H_0 \text{ if } G^2 > \chi^2_{\alpha, \, df} \] \[ \text{or if } p\text{-value} < \alpha \]

Find chi-square critical value: qchisq(1-α, df) or right-tail quantile.

# Critical value at α = 0.05, df = 5 critical_val = qchisq(0.95, df = 5, lower.tail = TRUE) # Or using right-tail area: critical_val_right = qchisq(0.05, df = 5, lower.tail = FALSE) # Example: 0.05 significance, df = 5 → 11.07
📌 For α = 0.05, reject if p-value ≤ 0.05 or \(G^2 \ge \chi^2_{0.05, df}\).

🎲 5. Worked Example: Die Fairness

\[ H_0: p_i = 1/6 \; (i=1,\dots,6) \]

Observed: 12,7,14,15,4,8 (n=60). Expected each =10.
\(G^2 = \frac{(2)^2}{10}+\frac{(-3)^2}{10}+ \dots = 9.4\), df=5, p=0.094 > 0.05 → fail to reject H₀.

# Die roll example die_obs = c(12, 7, 14, 15, 4, 8) die_exp_prob = rep(1/6, 6) chisq.test(die_obs, p = die_exp_prob) # X-squared = 9.4, df = 5, p-value = 0.094
🎲 Conclusion: insufficient evidence that die is unfair.

🏃 6. Practice: Exercise Habits

\[ H_0: p_{none}=0.60,\; p_{sporadic}=0.25,\; p_{regular}=0.15 \]

Survey (n=470): None=255, Sporadic=125, Regular=90. Expected: 282, 117.5, 70.5

# Health promotion example observed_exercise = c(255, 125, 90) hyp_prob = c(0.60, 0.25, 0.15) test_ex = chisq.test(observed_exercise, p = hyp_prob) test_ex # G^2 = (255-282)^2/282 + (125-117.5)^2/117.5 + (90-70.5)^2/70.5 ≈ 8.12 # p-value ≈ 0.0172 → Reject H₀ at α = 0.05
📊 Since p-value (≈0.017) < 0.05, distribution changed after campaign.

📉 7. Computing p-value (Right-tailed)

\[ p\text{-value} = P(\chi^2_{df} \ge G^2_{\text{obs}}) \]

Essential: Use pchisq(stat, df, lower.tail = FALSE)

# Example: test statistic G^2 = 9.4, df = 5 p_val = pchisq(9.4, df = 5, lower.tail = FALSE) print(p_val) # 0.094 # If G^2 = 15, df = 5 -> p = 0.0104 (significant)
🔁 Equivalent: 1 - pchisq(stat, df) but explicit lower.tail = FALSE is recommended.

📋 8. Assumptions & Sample Size

\[ \text{Expected frequency } E_i \ge 5 \text{ for each cell} \]

Random sampling, mutually exclusive categories, large enough expected counts.

# Check expected counts quickly expected_vals = sum(observed) * probs if (any(expected_vals < 5)) { warning("Low expected frequencies - consider combining categories") }
⚠️ If any E_i < 5, consider combining categories or exact tests.
