📊 Design of Experiments

Reference & Review Cards: CRD · RBD · ANOVA · Inferential procedures + R code
📈 One‑way ANOVA (CRD) F‑test
\[ F = \frac{MS_{\text{treatments}}}{MS_{\text{error}}} = \frac{SS_{\text{tr}}/(k-1)}{SS_{\text{error}}/(N-k)} \]
\(H_0: \mu_1 = \mu_2 = \dots = \mu_k\) vs \(H_a\): at least one difference
# CRD one-way ANOVA (balanced/unbalanced) model_crd = aov(response ~ treatment, data = crd_data) summary(model_crd) # post-hoc Tukey HSD TukeyHSD(model_crd)
🧱 RBD (no replication) Block + Treatment
\[ Y_{ij} = \mu + \tau_i + \beta_j + \varepsilon_{ij} \] \[ F_{\text{treat}} = \frac{MS_{\text{treat}}}{MS_{\text{error}}},\quad F_{\text{block}} = \frac{MS_{\text{block}}}{MS_{\text{error}}} \]
# RBD (each block contains each treatment once) model_rbd = aov(response ~ treatment + block, data = rbd_data) summary(model_rbd) # model assumptions: residuals and diagnostics plot(model_rbd)
🔬 Factorial Design Interaction
\[ Y_{ijk} = \mu + A_i + B_j + (AB)_{ij} + \varepsilon_{ijk} \] Test for interaction: \(H_0\): no interaction, \(F = MS_{AB}/MS_{error}\)
# two-way factorial with interaction model_fact = aov(response ~ factorA * factorB, data = factorial_data) summary(model_fact) # interaction plot interaction.plot(x.factor = factorA, trace.factor = factorB, response = response)
🔁 Repeated Measures Within-subject
\[ F = \frac{MS_{\text{time}}}{MS_{\text{error}}}, \quad \text{sphericity assumed} \] Subjects as random factor, controls individual differences.
# repeated measures ANOVA (using aov with Error) model_rm = aov(response ~ condition + Error(subject/condition), data = rm_data) summary(model_rm) # alternative: car::Anova(type=3) for more options
⚖️ Contrasts & Post‑hoc Multiple comparisons
\[ \psi = \sum_{i=1}^{k} c_i \bar{Y}_{i\cdot} , \quad SE(\psi) = \sqrt{MS_{error} \sum \frac{c_i^2}{n_i}} \] \(t\)-test for planned contrasts. Tukey HSD controls FWER.
# Custom contrast and TukeyHSD library(multcomp) contrast_matrix = rbind("New vs Old" = c(1, -1, 0)) model_crd = aov(yield ~ fertilizer, data = data) summary(glht(model_crd, linfct = contrast_matrix)) # Tukey HSD for all pairwise TukeyHSD(model_crd)
📦 RBD with Replicates Block × Treatment interaction?
\[ Y_{ijk} = \mu + \tau_i + \beta_j + (\tau\beta)_{ij} + \varepsilon_{ijk} \] Replication (\(p>1\)) allows interaction and pure error estimation.
# replicated RBD: multiple obs per block×treatment model_reprbd = aov(response ~ treatment * block, data = rep_rbd_data) summary(model_reprbd) # check interaction term significance # if interaction not significant, fit additive model
📐 ANOVA Diagnostics Normality · Homoscedasticity
\[ \text{Levene test: } H_0: \sigma^2_1 = \sigma^2_2 = \dots = \sigma^2_k \] Shapiro–Wilk for residuals normality.
# residual plots and homogeneity tests model = aov(response ~ treatment, data = data) par(mfrow = c(1,2)) plot(model, which = 1:2) # Levene's test (car package) library(car) leveneTest(response ~ treatment, data = data) shapiro.test(residuals(model))
📏 Effect Size (η², ω²) Practical significance
\[ \eta^2 = \frac{SS_{\text{treatment}}}{SS_{\text{total}}}, \quad \omega^2 = \frac{SS_{\text{tr}} - (df_{tr}) \cdot MS_{error}}{SS_{\text{total}} + MS_{error}} \]
# Eta-squared from ANOVA model = aov(yield ~ fertilizer, data = crop_data) summary(model) ss_total = sum((crop_data$yield - mean(crop_data$yield))^2) eta_sq = summary(model)[[1]]$`Sum Sq`[1] / ss_total # or use effectsize package library(effectsize) eta_squared(model)
🎲 Randomization CRD simulation
Simple random assignment: each unit has equal probability. Block randomization via `sample()`.
# Completely randomized assignment (balanced) units = 1:30 treatment = rep(c("A","B","C"), each = 10) assigned = sample(treatment) # random permutation # Block randomization (strata) library(blockrand) block_assign = blockrand(n = 30, num.levels = 3, levels = c("Control","Trt1","Trt2"))
🧠 Mixed Model (RBD) Block as random effect
\[ Y_{ij} = \mu + \tau_i + \beta_j + \varepsilon_{ij},\quad \beta_j \sim N(0, \sigma^2_{\text{block}}) \] Mixed models generalize RBD when blocks sampled from larger population.
library(lmerTest) # block as random factor mixed_model = lmer(response ~ treatment + (1 | block), data = rbd_data) anova(mixed_model) # F-tests with Satterthwaite summary(mixed_model)
